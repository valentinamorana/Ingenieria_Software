WardrobeFlow – Trabajo Práctico Integrador
Ingeniería de Software

Descripción del Sistema
WardrobeFlow es un sistema de gestión orientado a la administración de prendas mediante un modelo de suscripción.
Permite a los clientes acceder a un conjunto de prendas según el plan contratado, gestionar pedidos y controlar el stock disponible, asegurando una correcta rotación y seguimiento de las prendas.

Entidades Principales
Cliente: Usuario que contrata un plan de suscripción
Plan: Define la cantidad de prendas disponibles para el cliente
Prenda: Elemento del inventario con estado (disponible, en uso, etc.)
Pedido: Solicitud de prendas realizada por un cliente
Usuario: Actor del sistema (administrador, operador, etc.)
Bitácora: Registro de eventos y acciones dentro del sistema

Funcionalidades Principales
Gestión de clientes (alta, baja y modificación)
Administración de planes de suscripción
Control de stock de prendas
Creación y gestión de pedidos
Asignación de prendas según disponibilidad y plan
Seguimiento del estado de las prendas
Registro de acciones en bitácora
